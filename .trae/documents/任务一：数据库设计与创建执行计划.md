## 任务一：数据库设计与创建执行计划详情

### 步骤 1：连接 MySQL 数据库
- 使用命令行工具连接到 MySQL 服务
- 输入命令：`mysql -u root -p`
- 输入密码：`123456`

### 步骤 2：创建数据库
- 在 MySQL 命令行中执行创建数据库的 SQL 语句
- 数据库名称：`test_db`
- 执行命令：`CREATE DATABASE IF NOT EXISTS test_db;`

### 步骤 3：选择使用创建的数据库
- 执行命令：`USE test_db;`

### 步骤 4：设计并创建用户表
- 设计表结构，包含以下字段：
  - `id`：主键，自增
  - `name`：用户名，字符串类型
  - `email`：邮箱，字符串类型，唯一
  - `password`：密码，字符串类型
  - `created_at`：创建时间，时间戳类型
- 执行创建表的 SQL 语句

### 步骤 5：添加测试数据
- 向用户表中插入 3-5 条测试数据
- 执行插入数据的 SQL 语句

### 步骤 6：验证数据库操作
- 查看数据库列表，确认 `test_db` 已创建
- 查看用户表结构，确认字段设置正确
- 查看用户表数据，确认测试数据已插入成功

### 预期结果
- 成功创建数据库 `test_db`
- 成功创建用户表 `users`
