# API接口测试学习任务清单

## 项目概述
根据 project_rules.md 文件，本项目是一个简单的后端与数据库学习测试项目，使用 postman 作为前端测试工具。

## 技术栈
- 前端：postman工具（已安装）
- 后端：Node.js（全局已安装） + Express
- 数据库：MySQL96（全局已安装），用户名：root，密码：123456，端口：3306，

## 具体学习任务

### 任务1：数据库设计与创建
- [ ] 设计简单的数据库表结构（如用户表）
- [ ] 创建数据库(mysql服务我已经启动了)，直接使用mysql -u root -p连接数据库，之后再填写密码123456即可
- [ ] 创建数据表并添加测试数据

### 任务2：项目初始化与环境搭建
- [] 初始化 Node.js 项目（创建 package.json 文件）
- [] 安装 Express 框架
- [] 安装 MySQL 驱动（mysql2）
- [] 安装其他必要依赖（如 nodemon 用于开发时自动重启）

### 任务3：后端服务搭建
- [ ] 创建 Express 应用实例
- [ ] 配置数据库连接
- [ ] 实现基本的服务器启动与监听

### 任务4：API 接口实现
- [ ] 实现 GET 接口（如获取所有用户列表）
- [ ] 实现 POST 接口（如创建新用户）
- [ ] 实现 PUT 接口（如更新用户信息）

### 任务5：使用 Postman 测试 API

- [ ] 为每个 API 接口创建测试请求
